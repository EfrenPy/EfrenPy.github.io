## Project Overview

**Role:** Developer | **Type:** Personal Project | [Website](https://starter-pack.efrenrodriguezrodriguez.com/)

### Description

A bilingual (Spanish/English) static website designed to help newcomers starting to work at CERN. It provides comprehensive guides on legal and tax matters, social security, and technical setup, all organized in one accessible place.

### Features

- Bilingual support (Spanish and English) with full i18n system
- Legal and tax guidance for CERN employees with ties to Spain
- Technical tutorials including VS Code Remote setup and CERN tools
- Built-in search functionality across all articles
- Dark/light theme with system preference detection
- Responsive design optimized for all devices
